## 🚀 第一步：下载 ngrok（2 分钟）

### 1. 访问 ngrok 官网
https://ngrok.com/download

### 2. 下载 Windows 版本
点击 "Download for Windows"

### 3. 解压到桌面
将 `ngrok.exe` 解压到桌面

### 4. 注册账号
访问：https://dashboard.ngrok.com/signup
注册一个免费账号

### 5. 获取 authtoken
登录后访问：https://dashboard.ngrok.com/get-started/your-authtoken
复制你的 authtoken

### 6. 配置 authtoken
打开命令行（CMD），运行：
```bash
cd Desktop
ngrok config add-authtoken 你的authtoken
```

✅ ngrok 配置完成！

## 🚀 第二步：启动服务（1 分钟）

### 双击启动脚本
双击：`一键启动云端服务.bat`

会自动打开两个窗口：
1. **后端服务窗口** - 显示 API 日志
2. **ngrok 窗口** - 显示公网地址

### 复制 ngrok 地址
在 ngrok 窗口中找到：
```
Forwarding    https://abc123.ngrok.io -> http://localhost:8765
```

复制 `https://abc123.ngrok.io` 这个地址

✅ 服务启动完成！

## 🆘 遇到问题？

### 后端服务启动失败
- 检查 Python 是否安装
- 检查依赖是否安装（`pip install -r requirements.txt`）
- 检查端口 8765 是否被占用

### ngrok 连接失败
- 检查 authtoken 是否配置正确
- 检查网络连接
- 尝试重启 ngrok

### Streamlit 显示"后端服务未启动"
- 检查后端服务是否在运行
- 检查 ngrok 是否在运行
- 检查 API_URL 是否正确
