## 🎨 项目简介

这是一个基于 **Electron + Python** 的桌面应用，帮助你自动化求职流程：

### 核心功能
- ✅ **Boss直聘自动投递** - 批量投递岗位，节省时间
- ✅ **AI 简历分析** - 4个专业 AI Agent 分析简历
- ✅ **AI 生成求职信** - 个性化求职信，提高成功率
- ✅ **智能岗位搜索** - Boss直聘 + OpenClaw 双引擎
- ✅ **投递记录管理** - 完整的投递历史和统计
- ✅ **飞书通知** - 实时推送投递结果

### 技术栈
- **前端**: Electron + React + TypeScript + Ant Design
- **后端**: Python + FastAPI + Playwright
- **AI**: DeepSeek API（兼容 OpenAI）

## 📦 安装步骤

### 第一步：获取项目

如果你已经有项目文件夹，跳过这步。

```bash
# 方式 1: 使用 Git
git clone <项目地址>
cd ai-job-applier-desktop

# 方式 2: 直接解压下载的 ZIP 文件
```

### 第二步：安装后端依赖

```bash
# 进入后端目录
cd backend

# 安装 Python 依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器（用于自动化）
playwright install chromium
```

**如果安装很慢**，可以使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 第三步：配置环境变量

在 `backend` 目录下创建 `.env` 文件：

```bash
# 在 backend 目录下
notepad .env
```

填入以下内容：

```env
# DeepSeek API 配置（必需）
DEEPSEEK_API_KEY=sk-your-api-key-here
DEEPSEEK_BASE_URL=https://api.deepseek.com

# Boss直聘手机号（可选，可以在界面输入）
BOSS_PHONE=13800138000

# 飞书通知（可选）
FEISHU_WEBHOOK_URL=https://open.feishu.cn/xxx
```

**获取 DeepSeek API Key：**
1. 访问：https://platform.deepseek.com/
2. 注册/登录账号
3. 进入"API Keys"页面
4. 创建新的 API Key
5. 复制并粘贴到 `.env` 文件

### 第四步：安装前端依赖

```bash
# 回到项目根目录
cd ..

# 进入前端目录
cd electron

# 安装依赖
npm install
```

**如果安装很慢**，可以使用国内镜像：

```bash
npm install --registry=https://registry.npmmirror.com
```

### 方式 1：一键启动（推荐）

双击项目根目录下的 `start-dev.bat` 文件。

这会自动：
- ✅ 启动 Python 后端（端口 8765）

看到应用窗口弹出表示成功。

## 📖 功能使用

### 1️⃣ 登录 Boss直聘

**步骤：**

1. 点击左侧菜单 **"登录"**
2. 输入你的手机号
3. 点击 **"登录"** 按钮
4. 会弹出浏览器窗口
5. 在浏览器中：
   - 点击"获取验证码"
   - 输入收到的短信验证码
   - 点击"登录"
6. 登录成功后，浏览器会自动跳转
7. 回到应用，会显示"登录成功"

**注意：**
- 浏览器窗口不要关闭，保持在后台
- 登录后会自动保存 Cookie，下次启动不需要重新登录
- 如果遇到滑块验证码，程序会自动处理

---

### 2️⃣ 上传简历

**步骤：**

1. 点击左侧菜单 **"简历管理"**
2. 点击 **"上传简历"** 按钮
3. 选择你的简历文件（支持 PDF、Word）
4. 等待上传完成
5. 可以预览简历内容

**提示：**
- 简历内容会被提取为文本
- 用于后续的 AI 分析和求职信生成

---

### 3️⃣ AI 分析简历（可选但推荐）

**步骤：**

1. 点击左侧菜单 **"简历分析"**
2. 点击 **"开始 AI 分析"** 按钮
3. 等待 4 个 AI Agent 完成分析（约 1-2 分钟）：
   - 🎯 **职业分析师** - 分析你的职业背景和优势
   - 💼 **岗位推荐专家** - 推荐适合的岗位
   - 🎤 **面试辅导专家** - 提供面试建议
   - ✅ **质量审核官** - 审核简历质量
4. 查看分析结果

**作用：**
- 了解自己的优势和不足
- 获取岗位推荐
- 改进简历内容

---

### 4️⃣ 搜索岗位

#### 方式 A：Boss直聘搜索

**步骤：**

1. 点击左侧菜单 **"岗位搜索"**
2. 输入搜索条件：
   - **关键词**：如 "Python实习"、"前端开发"
   - **地点**：如 "北京"、"上海"
   - **最低薪资**（可选）：如 8000
3. 点击 **"搜索"** 按钮
4. 等待搜索结果（约 5-10 秒）
5. 查看岗位列表

#### 方式 B：OpenClaw 搜索（真实数据）

**步骤：**

1. 点击 **"OpenClaw 搜索"** 标签
2. 输入搜索条件
3. 点击 **"搜索"** 按钮
4. 等待搜索结果

**注意：**
- OpenClaw 需要单独安装（可选）
- 提供更真实的岗位数据
- 搜索速度较慢

### 5️⃣ 批量投递

**步骤：**

1. 在岗位搜索结果中，勾选要投递的岗位
   - 可以单个勾选

3. 自动跳转到 **"自动投递"** 页面
4. 在投递页面：
   - **输入简历内容**（粘贴你的简历文本）
   - **选择是否使用 AI 生成求职信**（推荐开启）
5. 点击 **"开始投递"** 按钮
6. 实时查看投递进度和日志

**投递过程：**
```
开始批量投递...
[1/10] 字节跳动 - Python开发实习生 ✅ 成功
[2/10] 腾讯 - 前端开发工程师 ✅ 成功
[3/10] 阿里巴巴 - Java开发 ❌ 失败: 今日沟通人数已达上限
...
✅ 批量投递完成！成功 8 个，失败 2 个
```

**注意：**
- 每个岗位投递间隔 3-6 秒（避免被限流）
- 如果失败，会显示失败原因
- 投递过程中不要关闭浏览器窗口

## ❓ 常见问题

### Q1: 登录失败怎么办？

**可能原因：**
- 手机号输入错误
- 验证码输入错误
- 网络问题

**解决方法：**
1. 检查手机号是否正确
2. 重新获取验证码
3. 检查网络连接
4. 关闭浏览器窗口，重新登录

---

### Q2: 投递失败怎么办？

**常见失败原因：**

1. **"今日沟通人数已达上限"**
   - Boss直聘每天有沟通人数限制
   - 解决：明天再试，或降低投递数量

2. **"该职位已关闭"**
   - 岗位已下线
   - 解决：跳过该岗位

3. **"已投递过"**
   - 重复投递
   - 解决：系统会自动跳过

4. **"未找到投递按钮"**
   - 岗位不支持在线沟通
   - 解决：跳过该岗位

---

### Q3: AI 分析失败怎么办？

**可能原因：**
- DeepSeek API Key 未配置
- API Key 无效或余额不足
- 网络问题

**解决方法：**
1. 检查 `.env` 文件中的 `DEEPSEEK_API_KEY`
2. 访问 https://platform.deepseek.com/ 检查余额
3. 检查网络连接
4. 重新生成 API Key

---

### Q4: 浏览器窗口关闭了怎么办？

**解决方法：**
1. 点击"登出"
2. 重新登录
3. 浏览器窗口会重新打开

**注意：**
- 投递过程中不要关闭浏览器窗口
- 可以最小化到后台

---

### Q5: 如何提高投递成功率？

**建议：**

1. **优化简历**
   - 使用 AI 分析功能
   - 根据建议改进简历

2. **使用 AI 求职信**
   - 开启"使用 AI 生成求职信"
   - 个性化内容更容易获得回复

3. **精准搜索**
   - 使用准确的关键词
   - 选择匹配的地点和薪资

4. **控制投递频率**
   - 每次投递间隔 3-6 秒
   - 每天投递不超过 50 个

5. **选择合适的时间**
   - 工作日 9:00-18:00
   - 避免深夜投递

---

### Q6: 如何避免被封号？

**注意事项：**

1. **不要频繁投递**
   - 每天不超过 50 个
   - 每次间隔 3-6 秒

2. **不要使用无头模式**
   - 保持 `headless=False`
   - 让浏览器窗口可见

3. **不要多设备同时登录**
   - 只在一个设备使用
   - 不要在手机和电脑同时登录

4. **模拟人类行为**
   - 程序已内置人类行为模拟
   - 包括鼠标移动、页面滚动、随机延迟

## ⚠️ 注意事项

### 投递限制

- **Boss直聘限制**：
  - 每天沟通人数有上限（约 50-100 人）
  - 达到上限后无法继续投递
  - 建议分批投递

### API 限流

- **DeepSeek API**：
  - 免费版有调用次数限制
  - 建议配置多个 API Key 轮换
  - 或升级到付费版

### 账号安全

- **不要分享账号**
- **不要频繁登录/登出**
- **不要在多个设备同时使用**
- **定期修改密码**

### 数据备份

- **定期导出投递记录**
- **备份简历文件**
- **保存重要的求职信**

### 法律合规

- **仅用于个人求职**
- **不要用于商业用途**
- **遵守平台规则**
- **尊重招聘方**

## 🔧 高级配置

### 自定义投递参数

编辑 `backend/automation/config.py`：

```python
# 投递间隔（秒）
random_delay_min = 3
random_delay_max = 6

# 每天最大投递数
max_apply_per_day = 50

# 公司黑名单
company_blacklist = ['外包', '劳务派遣', '猎头']

# 职位黑名单
title_blacklist = ['销售', '客服', '电话']
```

### 配置多个 API Key

编辑 `.env` 文件：

```env
# 多个 API Key，用逗号分隔
DEEPSEEK_API_KEY=sk-key1,sk-key2,sk-key3
```

程序会自动轮换使用。
