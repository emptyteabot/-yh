### 🚀 安装说明

#### 1. 安装应用

双击 `AI求职助手 Setup 1.0.0.exe` 进行安装：
- 安装过程约 10-30 秒
- 会自动创建桌面快捷方式
- 默认安装路径：`C:\Users\你的用户名\AppData\Local\Programs\AI求职助手\`

#### 2. 首次配置

安装完成后，首次启动需要配置：

1. **配置 DeepSeek API Key**
   - 在应用安装目录找到 `.env` 文件
   - 或者在应用内通过设置页面配置
   - 添加你的 DeepSeek API Key

2. **登录 Boss直聘**
   - 打开应用
   - 点击"登录"
   - 输入手机号
   - 输入验证码
   - 登录成功后会自动保存 Cookie

#### 3. 开始使用

**功能清单：**

1. **简历管理**
   - 上传简历（支持 PDF/Word）
   - 查看已上传的简历
   - 删除简历

2. **AI 简历分析**（4个专业 Agent）
   - 职业分析师：分析职业发展路径
   - 岗位匹配师：推荐匹配岗位
   - 面试教练：提供面试建议
   - 质量审核员：评估简历质量

3. **岗位搜索**
   - Boss直聘搜索：自动化搜索
   - OpenClaw 搜索：真实岗位数据
   - 筛选条件：关键词、地点、薪资、经验

4. **自动投递**
   - 批量投递：选择岗位批量投递
   - AI 生成求职信：每个岗位个性化
   - 实时进度：显示投递进度和结果

5. **智能投递**
   - 全自动：搜索 + 匹配 + 投递
   - AI 驱动：智能筛选匹配岗位
   - 飞书通知：投递结果实时推送

6. **投递记录**
   - 查看所有投递记录
   - 按状态筛选（成功/失败）
   - 导出为 Excel

### 🔧 常见问题

#### Q1: 安装后无法启动？

**解决方案：**
1. 检查是否有杀毒软件拦截
2. 右键 → 以管理员身份运行
3. 检查 Windows Defender 是否阻止

#### Q2: Python 后端启动失败？

**解决方案：**
1. 检查端口 8765 是否被占用
2. 查看应用日志（在安装目录）
3. 重启应用

#### Q3: Boss直聘登录失败？

**解决方案：**
1. 检查网络连接
2. 确认手机号正确
3. 检查验证码是否过期
4. 尝试重新登录

#### Q4: AI 分析失败？

**解决方案：**
1. 检查 DeepSeek API Key 是否配置
2. 检查 API Key 是否有效
3. 检查网络连接
4. 查看错误提示

#### Q5: 投递失败？

**可能原因：**
1. Boss直聘 Cookie 过期 → 重新登录
2. 岗位已关闭 → 跳过该岗位
3. 已投递过 → 系统会自动跳过
4. 网络问题 → 检查网络连接

### 📝 注意事项
