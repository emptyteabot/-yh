# 🚀 AI 求职助手 - 产品说明

## 这是什么？

这是一个**桌面应用程序**，帮你**自动投递简历到 Boss 直聘**，解放双手，提高求职效率。

### 一句话总结
**让 AI 帮你在 Boss 直聘上自动投递简历，每天可以投 50-100 个岗位，还能生成个性化求职信。**

---

## 🎯 核心功能

### 1. **Boss 直聘自动投递** ⭐⭐⭐⭐⭐
- 自动搜索岗位
- 自动点击"立即沟通"
- 自动发送打招呼消息
- 支持批量投递（一次投 10-50 个）

**效果**：
- 手动投递：每小时 5-10 个
- 自动投递：每小时 50-100 个
- **效率提升 10 倍！**

### 2. **AI 生成个性化求职信** ⭐⭐⭐⭐
- 分析岗位 JD（职位描述）
- 匹配你的简历
- 生成个性化打招呼消息
- 提高 HR 回复率

**示例**：
```
普通消息：您好，我对这个职位很感兴趣。

AI 生成：您好，我有 2 年 Python 开发经验，熟悉 Django 和 Flask，
看到贵公司的后端开发岗位与我的技能高度匹配，期待与您进一步交流。
```

### 3. **AI 简历分析** ⭐⭐⭐⭐
4 个专业 AI Agent 帮你分析简历：
- 🎯 职业分析师 - 分析你的优势和定位
- 💼 岗位推荐专家 - 推荐适合的岗位
- 🎤 面试辅导专家 - 提供面试建议
- ✅ 质量审核官 - 审核简历质量

### 4. **投递记录管理** ⭐⭐⭐
- 自动记录所有投递历史
- 统计成功率、回复率
- 避免重复投递
- 支持导出数据

## 🛠️ 技术架构

### 前端
- **Electron** - 桌面应用框架
- **React** - 用户界面
- **Ant Design** - UI 组件库

### 后端
- **Python FastAPI** - 后端服务
- **Playwright** - 浏览器自动化（模拟人工操作）
- **DeepSeek API** - AI 生成求职信

### 核心技术
- **反检测机制** - 避免被 Boss 直聘识别为机器人
- **人类行为模拟** - 随机延迟、鼠标移动、页面滚动
- **WebSocket 实时推送** - 投递进度实时显示

## 🎬 使用流程

### 第一次使用（10 分钟）

1. **启动应用**
   - 双击 `启动后端-8765.bat`
   - 双击 `启动前端-开发模式.bat`
   - 等待 Electron 窗口打开

2. **登录 Boss 直聘**
   - 点击左侧"登录"
   - 输入手机号
   - 输入验证码
   - 登录成功（只需登录一次，会保存状态）

3. **上传简历**
   - 点击"简历管理"
   - 上传 PDF 或 Word 简历
   - 系统自动提取文本

4. **开始投递**
   - 点击"Boss 自动投递"
   - 输入关键词（如"Python 开发"）
   - 输入城市（如"北京"）
   - 设置投递数量（建议先测试 5 个）
   - 点击"开始自动投递"

5. **查看进度**
   - 实时显示投递进度
   - 显示成功/失败状态
   - 显示 AI 生成的求职信

### 日常使用（5 分钟）

1. 打开应用（已登录，无需重新登录）
2. 点击"Boss 自动投递"
3. 输入关键词和城市
4. 点击"开始投递"
5. 等待完成

## ⚠️ 注意事项

### 1. 投递频率
- **建议**：每天投递 30-50 个
- **原因**：Boss 直聘有每日沟通人数限制
- **策略**：分 3 次投递（早中晚各一次）

### 2. 账号安全
- **不要**频繁投递（避免被限流）
- **不要**多设备同时登录
- **建议**：每次投递间隔 3-6 秒

### 3. 求职信质量
- **开启** AI 生成功能
- **效果**：回复率提升 2-3 倍
- **成本**：需要 DeepSeek API Key（很便宜，1000 次调用约 ¥2）

### 4. 法律合规
- **仅供个人使用**
- **不要用于商业目的**
- **遵守平台规则**
