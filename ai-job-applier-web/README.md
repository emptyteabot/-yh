## 🏗️ 技术架构

```
学生浏览器 (React)
    ↓ HTTP/WebSocket
FastAPI 后端
    ↓
任务队列 (内存/Redis)
    ↓
Playwright 自动投递
```

## 🚀 快速启动

### 第一步：安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 第二步：配置环境变量

在 `backend` 目录创建 `.env` 文件：

```env
DEEPSEEK_API_KEY=sk-your-api-key-here
SECRET_KEY=your-secret-key-change-in-production
```

### 第三步：启动服务

**方式 1：双击批处理文件**
- 双击 `启动后端.bat`
- 双击 `启动前端.bat`

**方式 2：命令行启动**

```bash
# 终端 1 - 启动后端
cd backend
python main.py

# 终端 2 - 启动前端
cd frontend
npm run dev
```

### 第四步：访问应用

打开浏览器访问：`http://localhost:5173`

## 📖 使用流程

### 1. 注册/登录
- 输入手机号
- 输入验证码（开发环境自动填充 123456）
- 自动注册并赠送 5 次免费投递

### 2. 开始投递
- 输入关键词（如"Python 实习"）
- 输入城市（如"北京"）
- 粘贴简历内容
- 点击"开始自动投递"

### 3. 实时查看进度
- WebSocket 实时推送投递进度
- 显示成功/失败状态
- 显示 AI 生成的打招呼消息

### 4. 升级套餐
- 免费次数用完后
- 点击"升级套餐"
- 选择合适的套餐（开发环境无需支付）
